JavaScript

It is a programming language used to build web applications. This record will contain the basics of JavaScript. JS is usually learned with HTML and CSS. It is easy to get started with. JavaScript is used with HTML and can be used to change or modify HTML elements.

<p id="demo" style="display:none">Hello JavaScript!</p>

<button type="button" onclick="document.getElementById('demo').style.display='block'">Click Me!</button>

This is a snippet to show an HTML attribute using JS. The Javascript code is scripted between the <script> </script> tags.

<script>console.log(2+2);</script>
